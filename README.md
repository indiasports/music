# General Music Player
Music player with general media controls. Plays local music files and displays info about the song and shows progess live.

### Features
- Play/Pause
- Previous/Next Song
- Repeat Functions (All/Single)
- Shuffle Mode
- Volume Control (TODO)

- Song Info (TODO)
  - Song Title
  - Album
  - Artist
  - Year
  - Album Cover
